# Vesper 系统架构

## 组件关系

```
┌─────────────────────────────────────────────────────┐
│                    浏览器 (HTTPS)                     │
│               https://c2.example.com                 │
└──────────────────────┬──────────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────────┐
│                  Vesper (Go 单二进制)                  │
│  ┌─────────────┐   ┌──────────────────────────────┐ │
│  │  静态前端     │   │        Gin API 服务           │ │
│  │  (Vue3 内嵌)  │   │  /auth  /api/*  WebSocket   │ │
│  └─────────────┘   └───────────┬──────────────────┘ │
└────────────────────────────────┼─────────────────────┘
                                 │ 本地 gRPC (127.0.0.1)
┌────────────────────────────────▼─────────────────────┐
│               Sliver Server (守护进程)                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────────────────┐ │
│  │ 监听器管理 │ │ 会话管理  │ │  载荷生成 (Go 工具链)  │ │
│  └──────────┘ └──────────┘ └──────────────────────┘ │
│                   :31337 (gRPC)                      │
│                   :53    (DNS)                       │
│                   :80/443 (HTTP/HTTPS C2)            │
└──────────────────────┬──────────────────────────────┘
                       │ 出网回调
┌──────────────────────▼──────────────────────────────┐
│                  目标主机 (Implant)                    │
│           Beacon / Session (mTLS/HTTP/DNS)           │
└─────────────────────────────────────────────────────┘
```

## 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 前端 | Vue 3 + NaiveUI + UnoCSS | SPA，编译为静态文件 |
| 后端 | Go + Gin | API 路由、认证、静态文件服务 |
| C2 引擎 | Sliver Server (gRPC) | 监听器、会话、载荷生成 |
| 通信 | gRPC (Vesper ↔ Sliver) | 本地 127.0.0.1:31337 |
| 嵌入 | `//go:embed all:dist` | 前端打包进二进制 |

## 数据流

```
用户操作 → Vue Router → API 请求 → Gin Handler → Sliver gRPC → Implant
                                        ↓
                              Sliver Operator Config
                              (~/.sliver/configs/)
```

- **认证**：Vesper 内置 `admin/admin123`，JWT token 24h 有效
- **Sliver 连接**：读取 `~/.sliver/configs/admin1_127.0.0.1.cfg`，含 token + mTLS 证书
- **静态文件**：生产模式从内嵌 `dist/` 读取，开发模式走 Vite 代理

## 启动流程

```
1. Sliver daemon 启动 (:31337)
2. 初始化 operator (admin1 / all permissions)
3. Vesper 启动
   ├── 连接 Sliver gRPC
   ├── 安装 Go 包装器 (兼容性)
   └── 监听 HTTP (:8088)
```
