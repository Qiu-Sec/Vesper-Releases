# 功能使用指南

## 总览

登录后进入总览页面，顶部卡片显示在线会话、监听器、凭据、战利品数量。下方快捷入口直达各功能模块。

## 监听器管理

通信管理 → 监听器

### 启动监听器

点击 `+ 启动监听器`，选择协议类型：

| 协议 | 端口示例 | 适用场景 |
|------|---------|----------|
| HTTP | 80/443 | 模拟 Web 流量 |
| HTTPS | 443 | 加密通信 |
| mTLS | 9090 | 双向认证 |
| DNS | 53 | DNS 隧道 |
| WireGuard | 51820 | 高性能隧道 |

- **监听地址**：`0.0.0.0` 表示所有接口，`127.0.0.1` 仅本机
- **域名**：HTTPS 监听器可绑定域名
- **持久化**：监听器会保留在 Sliver 中，重启后自动恢复

### 停止监听器

在监听器列表点击 `删除`，该监听器上的所有会话将断开。

## 木马生成

木马生成 → 生成新马

### 配置参数

| 参数 | 说明 |
|------|------|
| 名称 | 载荷文件名（不含扩展名） |
| 操作系统 | windows / linux / darwin |
| 架构 | amd64 / i386 / arm64 |
| 格式 | EXECUTABLE / SHARED_LIB / SERVICE / SHELLCODE |
| C2 地址 | `mtls://IP:PORT` 或 `https://DOMAIN` |

### 常见 C2 地址格式

```
mtls://192.168.1.100:9090
https://c2.example.com
http://10.0.0.5:8080
dns://c2.example.com
```

> ⚠️ Windows 本地生成可能失败（Go 版本兼容性），推荐用 Linux Vesper 交错编译 Windows 载荷。

### 构建记录

生成的载荷显示在列表中，可直接下载或查看详情。

## 主机管理

目标管理 → 会话管理

### 会话类型

| 类型 | 特征 |
|------|------|
| Session | 持久实时连接，延迟低 |
| Beacon | 定时回连，流量隐蔽（默认 60s 间隔） |

### 右键菜单

右键点击会话行，弹出操作菜单：

- **交互终端** — 打开 xterm 执行系统命令
- **文件浏览** — 远程文件管理（上传/下载/删除）
- **进程列表** — 查看目标进程
- **截图** — 截取目标屏幕
- **执行 BOF** — 内存执行 COFF 对象文件
- **端口转发** — 创建 SOCKS5/HTTP 隧道
- **重命名** — 修改会话显示名
- **Reconfigure** — 更改 Beacon 回连间隔/Jitter

## 交互终端

目标管理 → 交互终端（或从主机管理右键进入）

支持 xterm 全功能终端，常用命令：

```bash
whoami            # 当前用户
hostname          # 主机名  
ps                # 进程列表
netstat           # 网络连接
shell             # 弹出系统 shell
execute cmd.exe   # 执行命令（不交互）
```

文件浏览器在终端右侧，支持拖拽上传。

## 代理管理

隧道代理

通过已上线主机建立 SOCKS5 或 HTTP 代理：

```
1. 右键主机 → 端口转发
2. 选择 SOCKS5 或 HTTP
3. 指定代理端口
4. Vesper 本地监听该端口
```

配置浏览器/工具使用 `socks5://127.0.0.1:<端口>` 即可通过目标主机访问内网。

## AI Agent

AI Agent → 新建对话

支持 DeepSeek / OpenAI / Anthropic 等多模型。需在设置中配置 API Key。

功能：
- 自由对话
- 渗透测试辅助（漏洞分析、命令生成、报告编写）
- 会话上下文保留

## 凭据与战利品

数据收集 → 凭据查看 / 战利品浏览

自动从目标主机收集的凭据（密码哈希、token）和文件（截屏、下载文件）在此查看。

- **凭据**：支持按类型筛选（哈希、明文、Token）
- **战利品**：支持下载、预览文本文件
